# Инструкция по развертыванию на Beget

## 1. Подготовка билда

Билд уже готов в папке `dist/`. В нем содержатся:
- `index.html` - главная страница
- `assets/` - все JS, CSS и другие ресурсы
- `send-mail.php` - скрипт отправки писем

## 2. Загрузка на Beget

### Вариант А: Через FTP (FileZilla)
1. Подключитесь к FTP вашего хостинга
2. Перейдите в папку `public_html` или корневую папку вашего домена
3. Загрузите **все файлы** из папки `dist/`:
   - `index.html`
   - папку `assets/` со всем содержимым
   - `send-mail.php`

### Вариант Б: Через панель управления Beget
1. Войдите в панель управления Beget
2. Откройте Файловый менеджер
3. Перейдите в папку `public_html` или корневую папку домена
4. Загрузите все файлы из `dist/`

## 3. Настройка .htaccess для React Router

Создайте файл `.htaccess` в корне сайта (рядом с index.html) со следующим содержимым:

```apache
<IfModule mod_rewrite.c>
  RewriteEngine On
  RewriteBase /
  
  # Не перенаправлять реальные файлы
  RewriteCond %{REQUEST_FILENAME} !-f
  RewriteCond %{REQUEST_FILENAME} !-d
  
  # Перенаправить все на index.html (для React Router)
  RewriteRule . /index.html [L]
</IfModule>

# Включить кэширование для статических ресурсов
<IfModule mod_expires.c>
  ExpiresActive On
  ExpiresByType text/css "access plus 1 year"
  ExpiresByType text/javascript "access plus 1 year"
  ExpiresByType application/javascript "access plus 1 year"
  ExpiresByType image/jpeg "access plus 1 year"
  ExpiresByType image/png "access plus 1 year"
  ExpiresByType image/svg+xml "access plus 1 year"
</IfModule>
```

## 4. Проверка работы

После загрузки файлов:

1. **Откройте сайт** - должна отображаться главная страница
2. **Проверьте навигацию**:
   - Клик на "Программа" - должен прокручивать к разделу программы
   - Клик на "Результаты" - должен прокручивать к разделу результатов
   - Перейдите на `/privacy` - должна открыться страница политики конфиденциальности
   - Перейдите на `/terms` - должна открыться страница условий использования
3. **Проверьте форму**:
   - Заполните форму контактов
   - Отправьте форму
   - Письмо должно прийти на email@btbsales.ru

## 5. Возможные проблемы

### Форма не отправляется
- Убедитесь, что PHP работает на хостинге
- Проверьте, что функция `mail()` включена (обычно включена по умолчанию на Beget)
- Проверьте логи ошибок PHP в панели Beget

### Страницы /privacy и /terms возвращают 404
- Убедитесь, что создан файл `.htaccess` с правилами перенаправления
- Проверьте, что `mod_rewrite` включен (на Beget включен по умолчанию)

### Стили не загружаются
- Убедитесь, что папка `assets/` загружена полностью
- Проверьте права доступа к файлам (должны быть 644 для файлов, 755 для папок)

## 6. Структура файлов на сервере

```
public_html/
├── .htaccess           (создать вручную)
├── index.html          (из dist/)
├── send-mail.php       (из dist/)
└── assets/             (из dist/)
    ├── index-[hash].js
    ├── index-[hash].css
    └── ...
```

## 7. Настройка почты (опционально)

Если письма не приходят, можно настроить отправку через SMTP Beget:

1. В панели Beget создайте почтовый ящик
2. Измените `send-mail.php` для использования SMTP (если потребуется)
3. Обычно базовая функция `mail()` работает без дополнительных настроек
